Tags que iremos ver

Aqui você encontrará as principais tags que aprenderemos nesse curso de ProgramaçãoWEB

HTML5

A palavra HTML - Signifca HyperText Markup Language e nela é onde colocamos as nossas escritas e imagens para desenvolver um site.

CSS3

CSS3 - Cascading Style Sheets é um mecanismo onde você adiciona estilização ao seu documento, podendo fazer inline e também importando de um arquivo de fora

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web

BootStrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo

PHP

PHP é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web
